# Дневник погоды
**Автор:** Хатунцева Полина

**Описание:**
Простое приложение для ведения дневника погоды. Позволяет добавлять записи о погоде (дата, температура, описание), фильтровать их по дате и температуре. Данные сохраняются в файл JSON.

**Технологии:**
- Python 3.x.
- Tkinter (GUI).
- JSON (хранение данных).
- Git / GitHub.

## Как запустить приложение?
1. Установите Python 3.x.
2. Клонируйте репозиторий:
   ```
   git clone https://github.com/ВАШ_ЛОГИН/weather-diary.git
   ```
3. Перейдите в папку проекта.
4. Запустите:
   ```
   python main.py
   ```
5. Используйте интерфейс для добавления и просмотра записей.
